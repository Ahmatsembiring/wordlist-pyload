## 📋 Deskripsi

Repository ini adalah kumpulan **wordlist** dan **payload** yang digunakan untuk keperluan **security testing**, **penetration testing**, dan **ethical hacking**. Repository ini dibuat sebagai sumber daya pembelajaran dan referensi bagi para security researcher, pentester, dan siapa saja yang tertarik mempelajari cybersecurity.

### 🎯 Tujuan Pembuatan

Repository ini dibuat dengan beberapa tujuan:

1. **📚 Media Pembelajaran** - Sebagai bahan belajar untuk memahami berbagai jenis vulnerability dan cara exploitasinya
2. **🛠️ Tools Pentesting** - Menyediakan wordlist yang siap digunakan untuk security assessment
3. **🔍 Referensi Payload** - Kumpulan payload untuk berbagai jenis attack vectors
4. **🌱 Berbagi Pengetahuan** - Berkontribusi pada komunitas cybersecurity Indonesia
5. **⚡ Efisiensi Testing** - Mempercepat proses security testing dengan wordlist yang terorganisir

## 📁 Daftar Wordlist & Payload

Repository ini berisi berbagai kategori wordlist dan payload:

### 🔻 SQL Injection
- `sql-auth-bypass.txt` - Payload untuk bypass authentication
- `sql-error-based.txt` - Error-based SQL injection
- `sql-time-based.txt` - Time-based blind SQL injection

### 📂 Local File Inclusion (LFI)
- `LFI-payload.txt` - Basic LFI payloads
- `lfi-payload.txt` - Advanced LFI techniques

### 🎯 XSS (Cross-Site Scripting)
- `payloadxss.txt` - XSS payloads berbagai tipe

### 🔍 Directory Bruteforce
- `DirBuster-2007_directory-list-2.3-medium.txt` - Wordlist untuk directory enumeration

### 🔑 Password Lists
- `password.txt` - Common passwords
- `password-angka.txt` - Password kombinasi angka
- `jhon-the-ripper-username.txt` - Username wordlist

### 🔐 Hash & Encryption
- `hash2.txt` - Hash samples untuk testing
- `cipher.txt` - Cipher text samples

### 🌐 Web Reconnaissance
- `ipwebsite.txt` - IP address wordlist
- `dicc.txt` - Dictionary wordlist

### ⚔️ Attack Scripts
- `ddos.txt` - DDoS testing payloads
- `ddos-script.txt` - DDoS simulation scripts
